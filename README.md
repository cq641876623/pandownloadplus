# pandownloadplus
## 替代pandownload应用(根据pandownload原理)
简易的百度云盘下载器，原理根据pandownload而来，结合百度云盘外链的油猴脚本而来，项目结果简单，且粗糙，使用electron、websocket、aria2c组成、根据electron和aria2c的特性也可以开发出macos或android版本,但本人并没有mac,所以有能力者自取。
